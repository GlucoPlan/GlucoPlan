# GlucoPlan

Бесплатная программа для Windows — учёт питания и расчёт доз инсулина при диабете 1 типа.

---

## Возможности

- **Калькулятор** — КБЖУ, ХЕ, гликемическая нагрузка, расчёт дозы инсулина
- **NightScout / CGM** — автоподстановка сахара, тренд, поправка к дозе
- **Симулятор** — прогноз сахара на 4 часа, профили Фиасп/Новорапид/Хумалог
- **База продуктов** — импорт/экспорт CSV
- **История** — с копированием приёма обратно в калькулятор
- **Проверка обновлений** — баннер на вкладке Настройки

---

## Установка

### Готовый .exe

1. [Releases](../../releases/latest) → скачайте `GlucoPlan-vX.X.X-win64.zip`
2. Распакуйте, запустите `GlucoPlan.exe`

**Обновление:** замените только `GlucoPlan.exe`, файл `glucoplan.db` с данными сохранится.

### Из исходного кода

```bash
git clone https://github.com/your-username/GlucoPlan.git
cd GlucoPlan
pip install pillow
python main.py
```

Требования: Python 3.10+, Pillow.
